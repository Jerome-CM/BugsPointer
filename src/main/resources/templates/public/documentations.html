Pour télécharger le projet rendez-vous sur l’url : ...
Vous devez avoir un compte valide pour obtenir les différents fichiers

Implémenter les fichiers sur votre site

Pour profiter de BugsPointer sur votre site, il vous suffit de copier/coller les fichiers que vous avez téléchargé.
Placer les dans votre arborescence :

modalCss.css à placer dans votre dossier de style
modalControl.js à placer dans votre dossier js
modal.html à placer à la racine de votre site

Il vous faudra ensuite récuperer ces fichiers en indiquant le chemin dans vos fichiers.
Le lien ou le logo (modal.html) sont à inclure dans toutes les pages que vous voulez debugger.

Configuration

Renseigner votre clé de sécurité

Rendez-vous dans votre page Account pour trouver votre clé. Copier-la puis coller la dans le fichier modal.html dans le champ suivant, à la place de YOUR_KEY_HERE :

<input type=”hidden” name=”key” value=”YOUR_KEY_HERE”>
